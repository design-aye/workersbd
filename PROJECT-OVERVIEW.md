# WorkersBD Portal - Complete Project Overview

## 🎯 Project Summary

A production-ready, SEO-optimized, mobile-first bilingual (English/বাংলা) job portal for Bangladesh with comprehensive features for job seekers, workers, and employers.

## 🚀 Quick Start

### 1. Install Dependencies
```bash
cd workersbd-portal
npm install
```

### 2. Create Environment File
Create `.env.local`:
```env
NEXT_PUBLIC_SITE_URL=https://workersbd.com
NEXT_PUBLIC_GA_ID=G-XXXXXXXXXX
NEXT_PUBLIC_FB_PIXEL_ID=XXXXXXXXXXXXX
```

### 3. Generate Sitemap
```bash
npm run sitemap
```

### 4. Run Development Server
```bash
npm run dev
```
Visit: http://localhost:3000

### 5. Build for Production
```bash
npm run build
npm start
```

## 🎨 Customization Guide

### Change Brand Colors
Edit `tailwind.config.js`:
```javascript
colors: {
  primary: {
    500: '#0066cc',  // Your brand color
  }
}
```

### Add New District
Edit `lib/seo.js`:
```javascript
export const bangladeshDistricts = [
  { name: 'NewDistrict', namebn: 'নতুন জেলা', lat: XX.XXXX, lng: XX.XXXX },
];
```

### Add New Category
Edit `lib/seo.js`:
```javascript
export const jobCategories = [
  { slug: 'new-category', name: 'New Category', namebn: 'নতুন শ্রেণী' },
];
```
